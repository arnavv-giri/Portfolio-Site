# Contributing

Thank you for your interest in contributing to this portfolio project!

## 🚀 Getting Started

1. **Fork & Clone**:
   ```bash
   git clone https://github.com/arnavv-giri/portfolio.git
   cd portfolio
   ```

2. **Install Dependencies**:
   ```bash
   npm install
   ```

3. **Run Dev Server**:
   ```bash
   npm run dev
   ```
   Open [http://localhost:5173](http://localhost:5173).

4. **Build**:
   ```bash
   npm run build
   ```

## 📋 Contribution Types

| Type | Description |
|------|-------------|
| **🐛 Bug Fix** | Fix a glitch or broken animation |
| **✨ Feature** | Add a new section or interaction |
| **📖 Docs** | Improve README or guides |
| **🔧 Refactor** | Clean up code without changing behaviour |
| **🚀 Performance** | Optimise bundle size or scroll performance |

## 🏗 Code Style

- **JS**: ES modules, consistent formatting.
- **CSS**: Tailwind utilities first; custom only when needed.
- **HTML**: Semantic, accessible (alt texts, ARIA labels).
- **Commits**: Conventional format — `feat:`, `fix:`, `docs:`.

## 📝 Pull Requests

1. Create a feature branch: `git checkout -b feature/your-idea`.
2. Commit: `git commit -m "feat: describe your change"`.
3. Push: `git push origin feature/your-idea`.
4. Open a PR with a description and screenshots if visual.
